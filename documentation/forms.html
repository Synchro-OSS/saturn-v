Formulários

Formulários podem ser utilizados para páginas de cadastro, contato, pesquisa, etc.


Formulário Padrão

Code

<form class="sv-form">
    <label>
        <span>Name</span>
        <input type="text" name="name">
    </label>
    <label>
        <span>E-mail</span>
        <input type="text" name="email">
    </label>
    <label>
        <span>Gender</span>
        <select>
            <option value="">Please, select</option>
            <option value="M">Male</option>
            <option value="F">Female</option>
        </select>
    </label>
</form>
