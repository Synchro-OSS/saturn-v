Forms

A basic use of a form is using a class="sv-form" in a tag form. By default input(text, password, email), selects and textarea there are a full sized elements. 


<form class="sv-form">
    <label>
        <span>Name</span>
        <input type="text" name="name">
    </label>
    <label>
        <span>E-mail</span>
        <input type="text" name="email">
    </label>
    <label>
        <span>Gender</span>
        <select>
            <option value="">Please, select</option>
            <option value="M">Male</option>
            <option value="F">Female</option>
        </select>
    </label>
</form>


Inline forms

Inline forms use the same styles for elements, but elements has no full width and labels are floated. 


<form class="sv-form-inline">
    <label>
        <span>Name</span>
        <input type="text" name="name">
    </label>
    <label>
        <span>E-mail</span>
        <input type="text" name="email">
    </label>
    <label>
        <span>Gender</span>
        <select>
            <option value="">Please, select</option>
            <option value="M">Male</option>
            <option value="F">Female</option>
        </select>
    </label>
</form>


Forms & Grid

We can use a grid system in a form to have more controls of layout.
