Botões

O botão indica uma possível ação do usuário.


Tipos

Default button
Small button
Primary button
Info button
Danger button
Marged button

Code

<button class="sv-button">Default</button>
<button class="sv-button small">Small</button>
<button class="sv-button primary">Primary</button>
<button class="sv-button info">Info</button>
<button class="sv-button danger">Danger</button>
<button class="sv-button marged">Marged</button>


Combinando tipos

Podemos combinar mais de uma classe para um botão, como por exemplo:

Small

Podemos combinar a classe small com qualquer classe citada acima, no exemplo a seguir combinamos a classe primary com a classe small, onde a classe small reduz o tamanho do botão e a classe primary altera o estilo.

Aceita os termos acima citados?

Code

<button class="sv-button primary small">Aceitar</button>
<button class="sv-button small">Rejeitar</button>


Marged

Podemos combinar a classe marged com qualquer classe citada acima, no exemplo a seguir combinamos a classe info com a classe marged, onde a classe marged altera as margens do botão e a classe info altera o estilo.

Tem certeza que deseja excluir esse registro?

Code

<button class="sv-button info marged">Sim</button>
<button class="sv-button danger marged">Não</button>
