Botões

Botões são utilizados para realizar ações do usuário, em formulários por exemplo podem ser responsáveis por submetê-lo.


Conteúdo

  • Default button
  • Marged button
  • Primary button
  • Danger button
  • Info button
  • Full button


Default button

<>

Um simples botão padrão.

O botão pode ter dois tamanhos: default e small. 

<button class="sv-button">Default button</button>
<button class="sv-button small">Small button</button>


Marged button

<>

Veja o espaçamento default entre os botões.

Note a diferença no exemplo a seguir, utilizando a classe marged. 

    <button class="sv-button marged">Default button 1</button>
    <button class="sv-button marged">Default button 2</button>


Primary button

<>

Podemos utilizar a classe primary quando for necessário destacar alguma ação a ser realizada na página, normalmente esta classe é utilizada para indicar a ação primária da página. Ele pode ser utilizado com ambos os tamanhos de botão, basta acrescentar a classe primary, o mesmo vale para o espaçamento entre eles utilizando a classe marged. 

    <button class="sv-button primary">Primary button</button>
    <button class="sv-button primary small">Primary small button</button>
    <button class="sv-button primary marged">Primary marged button</button>


Danger button

<>

A classe danger é indicada para situações em que seja necessário ressaltar a importância de sua ação, como por exemplo ao deletar um registro. Assim como a classe primary, a classe danger também pode ser utilizada com ambos os tamanhos de botão e espaçamento. 

    <button class="sv-button danger">Danger button</button>
    <button class="sv-button danger small">Danger small button</button>
    <button class="sv-button danger marged">Danger marged button</button>


Info button

<>

A classe info pode ser utilizada com o mesmo principio da classe primary ou como referência a um link devido a sua cor azul. Assim como as classes anteriores a classe info também pode ser utilizada com ambos os tamanhos de botão e espaçamento.

    <button class="sv-button info">Info button</button>
    <button class="sv-button info small">Info small button</button>
    <button class="sv-button info marged">Info marged button</button>


Full button

<>

A classe info pode ser utilizada com o mesmo principio da classe primary ou como referência a um link devido a sua cor azul. Assim como as classes anteriores a classe info também pode ser utilizada com ambos os tamanhos de botão e espaçamento.

    <button class="sv-button full">Default full button</button>
    <button class="sv-button primary full">Primary full button</button>
    <button class="sv-button danger full">Default full button</button>
    <button class="sv-button info full">Default full button</button>
