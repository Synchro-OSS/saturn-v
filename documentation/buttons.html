Buttons

Botões são usados para a interação entre o usuário é a interface do sistema.


Default button

Um botão padrão, para uso geral. 

    <button class="sv-button default">Default button</button>


Buttons Styles

Diferentes estilos de botão.

  <button class="sv-button default">Default button</button>
  <button class="sv-button primary">Primary button</button>
  <button class="sv-button info">Info button</button>
  <button class="sv-button danger">Dange button</button>
  <button class="sv-button link">Link button</button>


Buttons Sizes

Saturn-V prove 2 tamanhos de botoẽs.

  <p>
    <button class="sv-button default">Default button</button>
    <button class="sv-button default small">Small button</button>
  </p>

Saturn-V também prove botões "full size".

  <button class="sv-button default full">Default full button</button>
  <button class="sv-button primary full">Primary full button</button>
  <button class="sv-button info full">Default full button</button>
  <button class="sv-button danger full">Default full button</button>


Button Margins

As vezes precisamos de mais espaço entre botões, nessa caso, usamos um botão com margens.

Default space:

Button with a margin: 

  <button class="sv-button default marged">Default button 1</button>
  <button class="sv-button default marged">Default button 2</button>


Disabled Button

Botões com o atributo disabled. 

  <button class="sv-button default" disabled>Button 1</button>
  <button class="sv-button primary" disabled>Button 1</button>
  <button class="sv-button info" disabled>Button 1</button>
  <button class="sv-button danger" disabled>Button 1</button>
