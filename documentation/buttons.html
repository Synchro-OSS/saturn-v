Botões

O botão indica uma possível ação do usuário.


Tipos

  • Default button
  • Small button
  • Primary button
  • Info button
  • Danger button
  • Marged button

Code

<button class="sv-button">Default button</button>
<button class="sv-button small">Small button</button>
<button class="sv-button primary">Primary button</button>
<button class="sv-button info">Info button</button>
<button class="sv-button danger">Danger button</button>
<button class="sv-button marged">Marged button</button>


Combinando tipos

Podemos combinar mais de uma classe para um botão, como por exemplo:

Small

Podemos combinar a classe small com qualquer classe citada acima.

  • Default button
  • Primary button
  • Info button
  • Danger button
  • Marged button

Code

<button class="sv-button primary small">Aceitar</button>
<button class="sv-button small">Rejeitar</button>


Marged

Podemos combinar a classe marged com qualquer classe citada acima.

Code

<button class="sv-button info marged">Sim</button>
<button class="sv-button danger marged">Não</button>
