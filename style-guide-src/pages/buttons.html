Primary button

<>

Podemos utilizar a classe primary quando for necessário destacar alguma ação a ser realizada na página, normalmente esta classe é utilizada para indicar a ação primária da página. Ele pode ser utilizado com ambos os tamanhos de botão, basta acrescentar a classe primary, o mesmo vale para o espaçamento entre eles utilizando a classe marged.

Danger button

<>

A classe danger é indicada para situações em que seja necessário ressaltar a importância de sua ação, como por exemplo ao deletar um registro. Assim como a classe primary, a classe danger também pode ser utilizada com ambos os tamanhos de botão e espaçamento.

Info button

<>

A classe info pode ser utilizada com o mesmo principio da classe primary ou como referência a um link devido a sua cor azul. Assim como as classes anteriores a classe info também pode ser utilizada com ambos os tamanhos de botão e espaçamento.
