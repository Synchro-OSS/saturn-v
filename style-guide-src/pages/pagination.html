--- title: Pagination icon: pagination pageTitle: Paginação ---

O menu de paginação é usado especialmente com o objetivo criar links para outras páginas.

Exemplos de uso

Podemos utilizar o menu de paginação de maneira simples utilizando a classe sv-pagination.

Anterior 1 2 3 4 5 Próximo

Code

{{#markdown}} ```html
Anterior 1 2 3 4 5 Próximo
``` {{/markdown}}

Classe active

Para fazer o controle da página atual, basta utilizar a classe active no item que representa a página.

1 2 3

Code

{{#markdown}} ```html
1 2 3
``` {{/markdown}}

Classe disabled

Podemos utilizar a classe disabled quando não for interessante mostrar todas as páginas disponiveis.

1 2 3 ... 9 10 11

Code

{{#markdown}} ```html
1 2 3 ... 9 10 11
``` {{/markdown}}

Combinando as classes

Podemos combinar todas as classes afim de tornar o menu de paginação completo.

1 2 3 ... 9 10 11

Code

{{#markdown}} ```
1 2 3 ... 9 10 11
``` {{/markdown}}
